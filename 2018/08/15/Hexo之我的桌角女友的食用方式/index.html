秀秀

通过使用一个名为 hexo-helper-live2d 的开源库，可以轻松的在自己的Hexo网站下贴上一只生猛可爱的萌妹子或主子：

什么是live2d

Live2d是11区宅男们开发出的虚拟女友育成技术，毕。（大雾）

就如知乎上所述的，live2d是通过连续的图像连接成一张相对随机实时变化的动画，可以对光标位置进行动作交互反馈。目前在GalGame上的应用比较多。

食用方式

在Hexo网站上，只需要导入 hexo-helper-live2d 库，依据口味搭配喜欢的素材包，3分钟加热即可食用：

npm install --save hexo-helper-live2d

  1. 在Hexo网站源码根目录上安装 hexo-helper-live2d 模块

    npm install --save hexo-helper-live2d

  2. （同一目录下）选择安装喜欢的素材包：

    npm install {packagename}
# 例如：npm install live2d-widget-model-z16

  1. 在网站配置文件或主题配置文件_config.yml中追加：
    live2d:
# 固定写法，具体用途参阅下方文档
enable: true
scriptFrom: local
pluginRootPath: live2dw/
pluginJsPath: lib/
pluginModelPath: assets/
tagMode: false
debug: false
model:
 # 这里根据已安装的素材包填写，如：live2d-widget-model-z16
 use: live2d-widget-model-素材包名
# 摆放位置
display:
 position: right
 width: 150
 height: 300
mobile:
 show: true
