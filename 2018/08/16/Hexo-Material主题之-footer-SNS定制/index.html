Hexo Material主题之 footer SNS定制

需求


老板：给我来一份这个👆

坑啊

Material主题预设了相当多的SNS入口，只需要在配置文件中填入对应的网址就会自动启用。然而这里没我想要的简书和博客园，怎么办？！（嗯..似乎这两个也不是什么SNS虽然..)

官方指南 中曾提及到配置文档中一个名为 footerico 的参数，用于指定footer中的SNS图标路径，然而在下载到的最新版1.5.6 Material 主题配置文件模板中并未发现这个字段..😓

经过多次测试发现，当前版本似乎并不支持自定义这个SNS图标路径，也不支持通过配置文件扩展想要追加的“SNS”入口（更直接一点，甚至连 themes/material/source/img/footer 这个目录也是假的，把里面的图标改了都不影响主题默认的SNS图标样式🤣

搬砖之路

机缘(JI) 巧合(ZHI) 之下，我选择了去主题仓库的 Issues 里面找找蛛丝马迹，果真找到了一条 footer 中 增加 segmentfault 入口 的功能合并报告，追查之下抵达了实现这个功能的代码提交部分，故事从此发生转机——

按照上古卷轴的指引，我们发现要实现自定义扩展 footer 的 SNS 入口非常简单，只需要下载一个图标，并更改3个文件的一小段代码（其实应该2个也应该足够了）👉

  1. 下载你想要的按钮图标的矢量文件，推荐一个网站 Iconfont-阿里妈妈矢量图标库
    使用GitHub账号登录即可下载，下载格式一定要选 SVG 格式，尺寸选择48像素

    用Atom或VScode打开已经下载的文件，找到里面的 svg 标签，完整摘取出来备用

  2. themes/material/layout/_partial/footer-left.ejs

    <!-- 这里追加一段需要添加的SNS入口，插入的位置就是在 footer 中的SNS出现顺序位置 -->
<!-- 这里以简书为例，所有 jianshu 关键字换成你需要的名字 -->
 <!-- Jianshu -->
 <% if(theme.sns.jianshu) { %>
     <a href="<%= theme.sns.jianshu %>" target="_blank">
         <button class="mdl-mini-footer--social-btn social-btn footer-sns-jianshu">
             <span class="visuallyhidden">Jianshu</span>
         </button><!--
  --></a>
 <% } %>

  3. themes/material/source/css/style.css

    <!-- 在文件末尾追加一段css样式，定义SNS图标 -->
<!-- jianshu 关键字换成你需要的名字 -->
.footer-sns-jianshu {
 background: url('data:image/svg+xml;utf8, <svg....>');
}
<!-- 上面的 svg 标签，使用第1步中新鲜摘取的 svg 整段覆盖上来 -->

  4. 将第3步完成的 style.css 内容全部copy，放到 Css在线解压缩/整理/格式化 - 在线工具 中进行css压缩，并贴到 themes/material/source/css/style.min.css 中（实际上这个才是最后会用到的 style 文件）

  5. 在主题配置文件 _config.yml 中，找到 SNS 部分，增加你需要的入口名称（跟上面的更改名字对应）

  6. 刷新博客页面，应该可以出来了
