QR Code Generator Online - 在线二维码生成器

起因

工作中经常要生成多个二维码，将数据扫描录入到设备，而且往往需要在多个二维码之间来回切换、随时回调历史记录。现有的在线工具或 App 不是塞满广告就是要繁杂的注册登录，难以找到满足需求的工具，干脆自己造一个。

功能

  • 输入文本或链接，即时生成二维码
  • 支持一键下载为 PNG 图片
  • 自动保存生成历史，刷新页面记录不丢失
  • 点击历史记录可快速切换查看
  • 支持 Ctrl/Cmd + Enter 快捷键生成
  • 浅色 / 深色主题自适应
  • 响应式布局，移动端可用

技术栈

  • Next.js 16 + React 19 + TypeScript
  • Tailwind CSS 4 负责样式
  • qrcode 库生成二维码（600x600px PNG）
  • 数据持久化基于 localStorage，完全客户端运行
  • 构建为纯静态文件，部署在 Vercel

整个项目没有后端服务，所有数据都保存在浏览器本地，轻量且隐私友好。
